#python Mnist1中网络结构

输入784--隐藏层1  512--隐藏层2  256--输出层 10
激活函数：ReLU（隐藏层+Softmax（输出层）
损失函数：交叉熵
权重初始化：He 初始化
优化方法：随机梯度下降（SGD）
